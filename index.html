Series也有不擅長的地方¶

  • Series缺乏延展性，一旦建立就無法改變長度
In [2]:
x = []
print(id(x), x)
x.append(1) # <- [1]
x.append(2) # <- [1, 2]
x.append(3) # <- [1, 2, 3]
x.append(4) # <- [1, 2, 3, 4]
print(id(x), x)

139910104334280 []
139910104334280 [1, 2, 3, 4]

In [7]:
x = pd.Series([])
print(id(x), list(x))
x = x.append(pd.Series([1])) # <- [1]
x = x.append(pd.Series([2])) # <- [1, 2]
x = x.append(pd.Series([3])) # <- [1, 2, 3]
x = x.append(pd.Series([4])) # <- [1, 2, 3, 4]
print(id(x), list(x))

139909627960512 []
139909627960344 [1, 2, 3, 4]

類別 Series List
特點 優異的運算速度 資料操作較靈活

  • 不要瞧不起List，List也是很厲害的！
